# vite-tailwind-storybook-ts-boilerplate

Quickly bootstrap a new project with this Boilerplate.

## This boilerplate contains all the tools 
- TypeScript 
- React 
- Vite 
- Storybook
- Vitest

## You can use this boilerplate to bootstrap your project.

Git clone this repo 

### Commands to run

> yarn Install

### To run storybook

> yarn storybook
